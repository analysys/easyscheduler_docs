1.0.3 release

Easy Scheduler Release 1.0.3

Easy Scheduler 1.0.3 is the fourth version in the 1.x series.

Enhanced：

  • [EasyScheduler-482]sql task mail header added support for custom variables
  • [EasyScheduler-483]sql task failed to send mail, then this sql task is failed
  • [EasyScheduler-484]modify the replacement rule of the custom variable in the sql task, and support the replacement of multiple single quotes and double quotes.
  • [EasyScheduler-485]when creating a resource file, increase the verification that the resource file already exists on hdfs

Repair:

  • [EasyScheduler-198] the process definition list is sorted according to the timing status and update time
  • [EasyScheduler-419] fixes online creation of files, hdfs file is not created, but returns successfully
  • [EasyScheduler-481] fixes the problem that the job does not exist at the same time.
  • [EasyScheduler-425] kills the kill of its child process when killing the task
  • [EasyScheduler-422] fixed an issue where the update time and size were not updated when updating resource files
  • [EasyScheduler-431] fixed an issue where deleting a tenant failed if hdfs was not started when the tenant was deleted
  • [EasyScheduler-485] the shell process exits, the yarn state is not final and waits for judgment.
